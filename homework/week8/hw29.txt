029. 

最大質數 

輸入一個數 x ，根據以下規則得到 k ，並找出 2 到 k 之間最大的質數。 
處理輸入請使用有限狀態機，不要使用陣列 

規則: 
1. 若 x 為正整數，且 2 <= x <= 1000，則 k = x 
2. 若 x = 0 或 x = 1，則 k = 1000 
3. 若 x 為負整數，則 k = x * -11 
4. 若 x 為正小數，則 k = 小數點前面的數 
5. 若 x 為負小數，則 k = 小數點後面的數 


規則範例: 
1. x = 50， k = 50 
2. x = 0， k = 1000 
3. x = -90， k = 990 
4. x = 66.22， k = 66 
5. x = -33.44， k = 44 

----------------------------------------- 
輸入說明： 
輸入一個數字。 

輸出說明： 
根據規則找出 k ，輸出 2 到 k 之間最大的質數。 
不合法的輸入則輸出Error!
